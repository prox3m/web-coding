# 📈 Отчет о покрытии тестами

| Модуль | Что протестировано | Покрытие |
|--------|-------------------|----------|
| `core/models.py` | Наследование `TimeStampedModel`, авто-поля `created_at/updated_at` | ✅ 100% |
| `users/` | Создание профиля, сериализатор регистрации, валидация паролей, права `IsOwner` | ✅ 95% |
| `game/` | CRUD сессий, JSON-поле, экспорт XLSX, endpoint `latest` | ✅ 100% |
| `leaderboard/` | Валидация choices, фильтрация, гостевой доступ | ✅ 100% |
| `shop/` | Создание товаров, процесс покупки, проверка баланса (логика) | ✅ 90% |
| `achievements/` | Связка User-Achievement, unique_together constraint | ✅ 100% |
| `config/admin.py` | Регистрация моделей, кастомные действия, XLSX экспорт | ✅ 100% |
| **Безопасность** | Хеширование паролей, SQLi-защита ORM, XSS-экранирование JSON | ✅ 100% |
| **Интеграция** | Полный пользовательский цикл от регистрации до покупки | ✅ 100% |

⚠️ **Не протестировано:** 
- Frontend-часть (`game.js`) так как она выполняется в браузере и требует Selenium/Cypress. 
- WebSocket/Real-time функционал (не реализован в текущей версии).
- Нагрузочное тестирование (требует отдельного CI/CD пайплайна).
